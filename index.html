CodeEdit is an exciting new code editor written entirely and unapologetically for macOS.

Develop any project using any language at speeds like never before with increased efficiency and reliability in an editor that feels right at home on your Mac.

CodeEdit is not meant to replace Xcode but rather supplement it for projects not intended for Apple platforms. We believe every type of developer deserves a great experience and excellent performance.
